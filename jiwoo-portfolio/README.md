## 🚀 GitHub Pages 배포 방법

1. 이 레포지토리를 GitHub에 Push
2. Settings → Pages → Source: `main` 브랜치 `/ (root)` 선택
3. Save → 배포 완료 (약 1~2분 소요)
4. `https://{username}.github.io/{repo-name}/` 접속
